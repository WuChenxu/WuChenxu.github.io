NexT主题从6.0开始只需要简单的配置就支持显示或者不显示更新时间了。

配置NexT主题

配置post layout参数

每篇post（md文件）的layout参数modified（或者updated）就是文章的更新时间。
如果modified与date相同，则只显示发表时间，不显示更新时间；如果modified与date不同，则date显示为发表时间，modified显示为更新时间；
