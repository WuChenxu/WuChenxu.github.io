通配符查找

find

常用option：
-name: name matching, can use regex.
-iname: like-name, but case insensitive.
-regex: regex匹配文件名（包含路径），egfind ./ -regex ".*/*hexo.py". -type: file type.d is directory, f is file.

grep

常用option:
-i（--ignore-case）: 忽略大小写
-r (--recursive）: 递归查找子目录
-n (--line-number): 打印行号
-l (--files-with-match): 只打印匹配的文件名
