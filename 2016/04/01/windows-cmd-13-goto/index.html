windows command(13):GOTO :label

发表于 2016-04-01 更新于 2017-07-16 分类于 windows cmd 阅读次数： Valine：

goto命令，跳转命令。

基本语法

跳转到标签行，并继续执行之后的语句。标签必须以:开头作为标签的指示。
标签名只可以使用常量，而不能使用变量。

条件 if或switch/case

注意:CaseA之后，如果不加goto end则会继续执行:CaseB之后的语句。

不过使用goto更加灵活，理论上for能实现的循环goto都能实现。

goto :EOF

直接跳转到文件结尾。
