windows command(5):TYPE

发表于 2016-03-25 更新于 2017-07-16 分类于 windows cmd 阅读次数： Valine：

TYPE命令用于显示文件内容，相当于bash的cat命令。

显示文件内容 TYPE filename

合并文件

直接合并 TYPE file1 file2 > mergefile

多次合并 type file >> mergefile

Refrenece:

cmd type
