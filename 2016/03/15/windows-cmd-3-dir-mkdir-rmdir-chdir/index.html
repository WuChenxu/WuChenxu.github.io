windows command(3):DIR MD RD CD

发表于 2016-03-15   |   更新于 2016-03-17   |   分类于 windows cmd   |  

DIR相当于bash里的ls，列出目录或文件。
MKDIR(MD)创建目录。
RMDIR(RD)删除目录。
CHDIR(CD)显示当前目录名(CD不跟参数时，兼有bash pwd命令功能)或者改变当前目录。
