windows command(9):FIND FINDSTR

发表于 2016-03-30   |   更新于 2016-06-28   |   分类于 windows cmd   |  

FIND和FINDSTR都是用来查找文件中的字符串的命令。FINDSTR支持通配符，是FIND的加强版本。

查找字符串

用空格隔开待查找的字符串"str1 str2"，在find中表示查找str1 str1整个字符串，比较符合直觉，
而在findstr中表示查找str或str2。findstr的待查字符串写成/C:"str1 str2"表示查找整个字符串.

查找的字符串中有"

在要查找的字符串里""转义成一个".
